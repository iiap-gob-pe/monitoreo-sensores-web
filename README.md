# Sistema de Monitoreo Ambiental con Sensores 🌡️

Sistema completo de monitoreo ambiental con ESP32, backend en Node.js y frontend en React.

## 📁 Estructura del Proyecto

- **sensor_monitoreo_api/** - Backend API (Node.js + Express + Prisma)
- **environmental-monitoring-web/** - Frontend (React)

## 🚀 Tecnologías

### Backend
- Node.js + Express
- Prisma ORM
- PostgreSQL
- CORS habilitado para ESP32

### Frontend
- React
- Recharts para gráficos
- Axios para peticiones HTTP
